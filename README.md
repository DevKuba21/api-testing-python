# API Automation Testing Project
Projekt zawiera testy automatyczne dla API serwisu To-Do (https://todo.pixegami.io/).

## Wykorzystane technologie
* **Python 3**
* **Requests** (do obsługi zapytań HTTP)
* **Pytest** (framework do uruchamiania testów)

## Opis testów
Skrypt weryfikuje następujące funkcjonalności:
* Poprawność kodów statusu (200 OK, 404 Not Found).
* Tworzenie nowych zadań (POST/PUT).
* Pobieranie i listowanie zadań.
* Aktualizację istniejących zadań.
